# AHC Problem Digests

AtCoder Heuristic Contest (AHC) の問題文要約や、人気投票の結果を集計・公開しているサイトです。

## コンテンツ

### 🗳️ 投票結果
AHCの過去問題に関する人気投票の結果をまとめています。
* [AHC001-063 投票結果](./ahc001-063/)

## このプロジェクトについて

本サイトは、AHCの各問題をGemini APIを用いて要約し、そのデータやアンケート結果をまとめるツール [ahc-problem-digests](https://github.com/blue-jam/ahc-problem-digests) によって生成されています。
