### <a id="ahc016"></a>AHC016 \- Graphorean (4票)

> 与えられたM個のグラフからノイズ混じりで頂点順序がシャッフルされたグラフHが与えられたとき、元のグラフを特定する問題。M個のグラフを事前に作成し、ノイズ後のHから元のグラフを予測することで得点を稼ぐ。グラフのサイズを小さくし、予測精度を高めることが重要。

- **wata**: 新しい設定&amp;理論的な考察が役立つ

### <a id="ahc033"></a>AHC033 \- Container Handling with Cranes (4票)

> N×Nのコンテナターミナルで、指定された順番でコンテナを搬出する問題。 1台の大クレーンとN\-1台の小クレーンを操作し、コンテナを掴む・離す・移動させる。 クレーンはコンテナを掴んでいると小クレーンは他のコンテナのマスに移動できない。 搬入口からコンテナが供給され、右端の搬出口から指定の番号順に搬出する。 搬出順の乱れや誤搬出を減らし、総ターン数も少なくすることが目標。

- **名無しさん**: 並列に動かすのを考えるのが難しくて、手でコーディングするのがめっちゃ大変だったから
- **名無しさん**: 人力で解いたときの動きを再現することに成功して上位に入れたのが嬉しかった
